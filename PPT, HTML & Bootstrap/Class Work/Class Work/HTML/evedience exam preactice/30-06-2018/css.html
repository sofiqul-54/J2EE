Hyper Text Markup Language (HTML) is the main markup language for creating web pages and other
information that can be displayed in a web browse.

HTML is written in the form of HTML elements consisting oftags enclosed in angle bracket (like
<html>), within the web page content. HTML tag most commonly come in pairs like <h1> and </h1>,
althougs some tags represent empty elements and so are unpaired, for example <img>. THe first tag in a
pair is the start tag, and the second tag is theend tag(they are also called opening tags and closing
tags). In between these tags web designers can add text, further tags, comments and other types of text-
based content.
